# 📦 Sistema de Controle de Estoque

> Projeto desenvolvido para a disciplina de Programação Orientada a Objetos (POO) do curso de Tecnologia em Análise e Desenvolvimento de Sistemas (TADS) no Instituto Federal do Rio Grande do Norte (IFRN).

## 📋 Sobre o Projeto

O **Sistema de Controle de Estoque** é uma aplicação construída em Java desenvolvida para gerenciar de forma eficiente o fluxo de mercadorias de um estabelecimento. O sistema aplica os conceitos fundamentais de POO para permitir um controle rigoroso sobre as operações do inventário.

## ✨ Funcionalidades

De acordo com os requisitos estabelecidos, o sistema contempla as seguintes operações principais:

- **Gerenciamento de Cadastros:**
  - Cadastro, visualização, edição e remoção de **Produtos**.
  - Gerenciamento de **Categorias** de produtos.
  - Cadastro e controle de **Fornecedores**.
- **Movimentação de Estoque:**
  - Registro de **Entrada** de produtos (compras/abastecimento).
  - Registro de **Saída** de produtos (vendas/baixas).
- **Monitoramento:**
  - **Alertas de Estoque Mínimo**: Sistema de notificação (visual ou via terminal) quando um produto atinge ou ultrapassa o limite crítico estabelecido, indicando a necessidade de reposição.

## 🛠️ Tecnologias e Conceitos Utilizados

- **Linguagem:** Java
- **Paradigma:** Programação Orientada a Objetos (Classes, Objetos, Encapsulamento, Herança, Polimorfismo, Coleções)
- **Organização:** Estrutura de pacotes lógica (ex: `entities`, `services`, `application`, baseada em boas práticas).
